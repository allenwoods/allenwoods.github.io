使用Vagrant打造跨平台Python环境

发表于 2015-10-16   |   分类于 技术

假期回家只能带着Windows的笔记本，但Python在Windows下诸多不便。Vagrant可以利用虚 拟机,实现在不同的操作系统下共享开发环境,简单记录下整个配置过程。

1 安装Prerequisite

  • Virtualbox
  • Vagrant
  • Putty (Windows系统，如无cygwin必须安装)

2 配置Vagrant

安装上述软件后(可能需要重启计算机), 进入命令行(Linux/OSX的shell或是Windwos下的 PowerShell).

2.1 创建安装目录

首先确定安装目录,

进入安装目录 D:\DataScienceToolbox.

2.2 安装虚拟镜像

由于国内网络的原因, 下载虚拟镜像文件速度奇慢, 建议使用其他方式下载。 假设下载后文件在当前安装目录下, 文件名为 dst.box 首先执行以下命令

vagrant box add <环境名> <文件名> 命令将 dst.box 导入了 data-science-toolbox/dst 配置内。

至此Vagrant就算装好了， 执行 vagrant up 启动虚拟机， vagrant ssh 登陆环境 （如出错需要使用putty登陆）。

3 配置Python

3.1 安装Anaconda

Anaconda 是一个优秀的Python发型版，内置了完整的科学计算库，可以直接下载安装。 安装时注意，Vagrant的共享目录默认设置在安装目录内，虚拟机环境内使用 cd /vagrant 切换至该 目录，可以读取宿主机器拷入的文件。将Anaconda安装文件(.sh文件)导入安装即可

3.2 配置Ipython（Jupyter）

在宿主机器的浏览器中输入 http://localhost：8888 进入Ipython。
